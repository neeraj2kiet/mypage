100% Height Elements

Just a simple (and ugly) little area to play with 100% heights.

More

height:100% wierd

The default is height:auto, so html and body must have height:100% set.

Elements nested deeper than one level into the body with height: 100% can overflow. Uncomment the "section" tag to see this.

This article has min-height: 100% set to allow content to flow to greater than 100% of the viewport height.

Setting height or min-height to 100vh is a simple way of creating full height elements without setting explicit heights on parents and worrying about nesting. (Uncomment min-height: 100vh in "div, article" css, comment out all other heights and see caniuse for browser support).
